Flat Level-1 multi-line field — same focus treatment as `Input` (2px accent border + 4px soft glow halo).

```jsx
<Textarea label="Bio" rows={4} placeholder="A few words…" />
<Textarea label="Notes" error="Too long" />
```

Pass `label`, `hint`, `error`, `rows`. All standard textarea attributes pass through. Resizes vertically.
